Features

📄

PDF & Image Support

Accepts PDF, PNG, JPG, and JPEG files. Works with native text PDFs and scanned documents via OCR.

🔍

Intelligent Parsing

Extracts invoice number, dates, vendor, tax number, subtotal, tax, total, currency, payment method, and line items.

🌐

Bilingual: EN & AR

Regex patterns support both English and Arabic invoice layouts including VAT, ضريبة القيمة المضافة, and more.

⚡

Fast & Lightweight

No heavy AI model required for basic extraction. Sub-second response for most text-based invoices.

🔒

Secure

API key authentication. Files are never stored permanently. Works with RapidAPI proxy headers.

📊

Confidence Score

Every response includes a confidence score (0–100) so you know how reliable the extraction is.

Example Request

cURL
Python
curl -X POST https://your-api.com/v1/invoices/extract \
  -H "X-API-Key: your-api-key" \
  -F "file=@invoice.pdf"
